# Password Master - 密码生成器

一个安全、快速、易用的在线密码生成工具，所有密码在本地生成，保护您的隐私安全。

## ✨ 功能特性

### 核心功能
- 🔐 **安全密码生成**：使用 Web Crypto API 生成加密安全的随机密码
- 📏 **自定义长度**：支持 4-128 位密码长度设置
- 🎯 **字符类型选择**：
  - 大写字母 (A-Z)
  - 小写字母 (a-z)
  - 数字 (0-9)
  - 特殊字符 (!@#$%^&*等)
- 🚫 **排除选项**：
  - 排除相似字符（0, O, l, I）
  - 排除歧义字符

### 高级功能
- 💪 **密码强度评估**：实时显示密码强度（弱/中/强/极强）和评分（0-100分）
- 📋 **一键复制**：快速复制密码到剪贴板
- 📦 **批量生成**：支持一次生成多个密码（最多50个）
- 💾 **导出功能**：将批量生成的密码导出为文本文件
- 📜 **历史记录**：本地保存最近生成的密码（最多50条）
- 🎨 **主题切换**：支持亮色/暗色主题
- 🎭 **预设模式**：快速选择常用密码模式

### 用户体验
- 📱 **响应式设计**：完美支持桌面端和移动端
- ⚡ **实时更新**：配置改变时自动重新生成密码
- 🎯 **简洁界面**：现代化、直观的用户界面
- 🔒 **隐私保护**：所有操作在客户端完成，不上传任何数据

## 🚀 快速开始

### 本地运行

1. 克隆或下载项目
```bash
git clone <repository-url>
cd password-master
```

2. 直接在浏览器中打开 `index.html` 文件

或者使用本地服务器：

```bash
# 使用 Python
python -m http.server 8000

# 使用 Node.js (需要安装 http-server)
npx http-server

# 使用 PHP
php -S localhost:8000
```

3. 在浏览器中访问 `http://localhost:8000`

### 部署

这是一个纯静态网站，可以部署到任何静态托管服务：

- **GitHub Pages**
- **Netlify**
- **Vercel**
- **Cloudflare Pages**
- 任何支持静态文件的 Web 服务器

## 📖 使用说明

1. **生成密码**：
   - 调整密码长度滑块
   - 选择需要的字符类型
   - 点击"生成密码"按钮或使用自动生成

2. **复制密码**：
   - 点击密码显示框右侧的复制按钮
   - 或直接点击密码显示框

3. **批量生成**：
   - 点击"批量生成"按钮
   - 输入要生成的密码数量（1-50）
   - 可以单独复制每个密码或导出全部

4. **查看历史**：
   - 在"密码历史"区域查看最近生成的密码
   - 可以复制历史记录中的密码
   - 可以关闭历史记录功能以保护隐私

5. **切换主题**：
   - 点击页面底部的主题切换按钮
   - 支持亮色和暗色两种主题

## 🛠️ 技术栈

- **HTML5**：语义化标记
- **CSS3**：现代化样式，CSS 变量支持主题切换
- **JavaScript (ES6+)**：原生 JavaScript，无框架依赖
- **Web Crypto API**：加密安全的随机数生成
- **LocalStorage API**：本地存储用户设置和历史记录

## 🔒 安全说明

- ✅ 所有密码在浏览器本地生成，不会上传到服务器
- ✅ 使用 Web Crypto API 确保随机性安全
- ✅ 不收集、不存储用户的任何个人信息
- ✅ 历史记录仅保存在本地浏览器中
- ✅ 支持 HTTPS 部署，确保传输安全
